# soul-harbor

<font size = 6>**介绍**</font>

./Back-end：后端相关文件

./Front-end: 前端相关文件

<font size = 6>**使用**</font>

<font size = 4>**后端**</font>

安装依赖：`cd ./Back-end npm install`

构建后端dist文件夹：`npm run build`

开启后端本地服务：`npm run watch`

后端本地服务热更新：`npm run watchDev`

<font size = 4>**前端**</font>

安装依赖：`cd ./Front-end npm install`

开启本地前端服务：`npm run start`

构建前端dist文件夹：`npm run build`
